# 🎬 Movie Recommendation System

This is a Movie Recommendation Web App built using Python and Flask.

## 🚀 Features
- Recommend similar movies using ML
- Clean UI with posters
- Fast and offline support

## 🛠️ Tech Stack
- Python
- Flask
- Pandas
- Scikit-learn

## ▶️ How to Run

1. Clone the repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py

## 📸 Output
Displays recommended movies with posters.
